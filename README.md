# 🎬 Movie Ticket Booking System

A simple Python program that calculates the final price of a movie ticket based on user details like age, seat type, show time, and discounts.

---

## 📋 Features

- ✅ Age-based ticket eligibility check
- 🌙 Evening show eligibility for users 21+
- 💳 Membership discount system
- 💰 Extra charges for weekends/evening shows
- 🪑 Seat-type-based service charges
- 🧮 Final price calculation

---

## 🛠️ Technologies Used

- **Python 3**

---

## 🚀 How to Run

1. Make sure Python is installed on your system
2. Clone or download this repository
3. Open terminal/command prompt in the project folder
4. Run the command:
   ```bash
   python ticket_booking.py
